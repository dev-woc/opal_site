Technical Implementation Details

G2P Converter: Uses CMU Pronouncing Dictionary with rule-based fallback. Production-ready for NVIDIA NeMo G2P-Conformer integration.
Heteronym Handler: Context-aware disambiguation for 8 common heteronyms including read, lead, bow, tear, live, wind, close, present.
Decodability Checker: Calculates percentage of decodable words based on allowed phonemes and sight words. Enforces 95% minimum threshold.
Orthographic Rules: Validates 1-1-1 doubling rule, Y-to-I changes, and silent E patterns with detailed explanations.
API Endpoints:
  • POST /api/phonetic/g2p-convert - G2P conversion
  • POST /api/phonetic/analyze-heteronym - Heteronym analysis
  • POST /api/phonetic/check-decodability - Decodability validation
  • POST /api/phonetic/validate-orthographic - Rule validation
  • GET /api/phonetic/stats - System statistics
