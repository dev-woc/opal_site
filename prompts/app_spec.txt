# Agentic Literacy Pipeline App

An agentic pipeline designed to synthesize Wilson Reading System (WRS), Orton-Gillingham (OG), UFLI Foundations, and FCRR into a cohesive system for generating interactive digital and high-fidelity print materials. 

## Tech Stack

* Orchestration: LangChain or CrewAI 


* Generation: GPT-4o with strict RAG constraints, Lexial databases 


* Phonetic Auditing: NVIDIA NeMo (G2P-Conformer), CMU Pronouncing Dictionary 


* Print & Layout: Python (ReportLab, WeasyPrint), Canva API, MagicSchool 



## Features (5 total)

### 1. Multi-Agent Orchestration

* "Curriculum Planner" decomposes learning objectives into structured lesson plans. 


* "Narrative Architect" generates engaging stories using strictly allowed phonemes. 


* "Diagnostic Specialist" analyzes student errors to suggest prescriptive teaching focuses. 


* Agents share a "scratchpad" or conversation history for unified state management. 



### 2. RAG-Powered Pedagogical Grounding

* Uses Retrieval-Augmented Generation (RAG) to ensure outputs are grounded in validated science. 


* Retrieves specific rules from Structured Literacy documentation rather than relying on broad LLM knowledge. 


* Pulls required word lists and activity data directly from the FCRR resource database. 



### 3. Automated Phonetic Verification

* Translates text into phonetic representations using Grapheme-to-Phoneme (G2P) conversion. 


* Distinguishes heteronyms by decomposing every word into constituent phonemes. 


* Maintains a 95% decodability rate to prevent students from guessing or using cueing systems. 


* Enforces orthographic rules, validating transformations like the 1-1-1 doubling rule or Y-to-I changes. 



### 4. Digital Interactivity & Gamification

* Virtual blending boards where students drag letter tiles and receive high-quality Text-to-Speech (TTS) auditory feedback. 


* Mobile haptic tapping that provides subtle vibrations to replicate multisensory sound segmenting. 


* Adaptive gamification loops that adjust difficulty in real-time, such as triggering multisyllabic "boss levels". 



### 5. Programmatic Print Generation

* Automates print production to support handwriting practice (encoding) and reduce screen fatigue. 


* Utilizes ReportLab for pixel-precise layout of word cards, syllable cards, and sound mapping boxes. 


* Interfaces with Canva's API to suggest layouts and generate visually engaging worksheet templates. 



## Design Guidelines

* Provide specific, instant feedback (e.g., identifying exact phonetic errors) rather than generic prompts. 


* Incorporate multi-modal cues including visuals, audio, and interactive tasks. 


* Ensure multi-modal accessibility by automatically enlarging text, increasing line spacing, or providing high-contrast modes for severe dyslexia. 


* Utilize the FCRR color-coded structure for activities (e.g., Purple for Phonological Awareness, Orange for Phonics, Green for Vocabulary).

## Success Criteria

* Texts successfully maintain a strict 95% decodability rate. 


* The system is fully observable, logging every AI decision (like target word selection or font size) for teacher review. 


* Reduces the educator "prompting gap" through "Flipped Interaction" patterns via simple drop-down menus. 


* Small instructional AI teams (under 7 agents) operate without conflicting assumptions.
